# HabitCare CLI

Aplicação em linha de comando para controle de hábitos diários, auxiliando na organização e manutenção de rotinas saudáveis.


## Solução
O sistema permite:
- adicionar hábitos
- listar hábitos
- concluir hábitos

## Público-alvo
- estudantes
- idosos
- pessoas com rotina intensa

## Tecnologias
- Python
- Pytest
- Ruff
- GitHub Actions

## Instalação
```bash
pip install -r requirements.txt
```

## Execução
```bash
python src/main.py
```
## Testes
```bash
pytest
```
## Lint
```bash
ruff check .
```
